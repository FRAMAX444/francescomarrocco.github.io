About me

Structured like a profile, readable like a feed.

Questa versione è pronta per GitHub Pages perché è un unico index.html con navigazione interna via hash (#home, #about, #works). Nessun framework, nessun build obbligatorio.

Le informazioni di queste sezioni vengono lette da una struttura dati interna al file, così puoi aggiungere nuove voci facilmente senza toccare la UI.

How to edit
Cerca nel file le costanti ABOUT_DATA e PUBLICATIONS_DATA. Aggiungi un oggetto per ogni esperienza o education e il sito genera automaticamente le card, ma il JSON non viene mostrato nella UI.
